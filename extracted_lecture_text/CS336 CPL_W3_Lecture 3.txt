▪ Logic, the science of reasoning and proof, has existed 
since the time of the ancient Greek philosophers. 
▪ Mathematical, or symbolic, logic, the theory of 
mathematical proofs, began with the work of George 
Boole and Augustus De Morgan in the middle of the 
1800s. 
▪ Since then, logic has become a major mathematical 
discipline.

--- PAGE 3 ---
▪ Logic is not concerned with what is true. Logic is the 
study of what follows from what. What conclusion 
follow from a set of premises.
▪ It can be defined as study of principles of correct 
reasoning.
▪ The main thing we study in logic are principles 
governing the validity of arguments and check that 
whether certain conclusion follows from some given 
assumption.

--- PAGE 4 ---
▪ Logic programming is a declarative programming 
paradigm based on formal logic.
▪ programs are constructed as sets of facts and rules 
rather than imperative instructions.
▪ It focuses on defining "what" the problem is rather 
than "how" to solve it, utilizing automated reasoning 
to deduce answers to queries.

--- PAGE 5 ---
▪ In normal programming, like Python or Java, we 
usually tell the computer how to solve a problem.
▪ For example, if we want to know who is a 
grandparent, we write steps:
1) look at parent relationships
2) check whether one parent leads to another
3) return the result
5
LOGIC PROGRAMMING 

--- PAGE 6 ---
▪ Logic programming tries to flip this idea.
▪ Instead of writing the steps, we write:
• facts: things that are true
• rules: relationships between truths
• queries: questions we want the system to answer
▪ So the programmer mostly describes what must be 
true, and the system tries to prove it.

▪ To match statements that contain variables, we must set the 
variables equal to terms so that the statements become identical 
and can be canceled from both sides.
▪ This process of pattern matching to make statements identical is 
called unification, and variables that are set  equal to patterns 
are said to be instantiated.
▪ Unification is the process by which variables are instantiated, or 
allocated memory and assigned values, so that patterns match 
during resolution. It is also the process of making two terms the 
same in some sense

▪ Prolog is the most widely used language inspired by 
logic programming research.
▪ It is a declarative language.
▪ Prolog is used for solving problems that involve objects 
and relationships between objects.
▪ A Prolog program can be seen as a relational 
database containing facts as well as rules.
54
PROLOG

--- PAGE 55 ---
▪ Prolog language basically has three different elements 
1) Facts: It is predicate that is true
• For example, if we say, “Tom is the son of Jack”, then this is 
a fact. 
2) Rules: They are extinctions of facts that contain 
conditional clauses. To satisfy a rule these conditions 
should be met. 
• For example, if we define a rule as: 
 grandfather(X, Y) :- father(X, Z), parent(Z, Y) 
• This implies that for X to be the grandfather of Y, Z 
should be a parent of Y and X should be father of Z.
